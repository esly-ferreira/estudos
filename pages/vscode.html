Extensões Open Settings (JSON)

VSCode

Auto Close Tag

https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-close-tag

A extensão Auto Close Tag do VS Code fecha automaticamente tags HTML, XML e similares assim que você abre uma, agilizando o desenvolvimento e evitando erros de sintaxe.

Auto Complete Tag

https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-complete-tag

A extensão Auto Complete Tag do VS Code completa automaticamente a tag de fechamento enquanto você digita a de abertura, prevendo a tag correta sem precisar digitar manualmente, o que acelera a escrita de HTML, XML e JSX.

Auto Import

https://marketplace.visualstudio.com/items?itemName=steoates.autoimport

A extensão Auto Import do VS Code adiciona automaticamente as instruções de importação no topo do arquivo ao usar funções, classes ou componentes de outros módulos, agilizando o desenvolvimento em JavaScript, TypeScript e frameworks como React.

Auto Rename Tag

https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag

A extensão Auto Rename Tag do VS Code renomeia automaticamente a tag de fechamento quando você altera a tag de abertura (e vice-versa) em arquivos HTML, XML, JSX e similares, garantindo que elas fiquem sempre sincronizadas sem você ter que editar manualmente as duas.

Code Runner

https://marketplace.visualstudio.com/items?itemName=formulahendry.code-runner

A extensão Code Runner do VS Code executa rapidamente trechos de código de várias linguagens direto no editor, sem precisar configurar projetos ou terminais, facilitando testar scripts e snippets de forma instantânea.

Color Highlights

https://marketplace.visualstudio.com/items?itemName=naumovs.color-highlight

A extensão Color Highlights do VS Code destaca visualmente no código as cores definidas em formatos como HEX, RGB, HSL, mostrando um preview da cor diretamente no editor, facilitando a identificação e edição rápida das cores usadas.

Icons

https://marketplace.visualstudio.com/items?itemName=tal7aouy.icons

A extensão Icons do VS Code adiciona ícones personalizados às pastas e arquivos no explorador, facilitando a identificação visual dos tipos de arquivos e melhorando a navegação no projeto

Image Preview

https://marketplace.visualstudio.com/items?itemName=kisstkondoros.vscode-gutter-preview

A extensão Image Preview do VS Code permite visualizar imagens diretamente no editor ou ao passar o mouse sobre o nome do arquivo, sem precisar abrir um programa externo, acelerando a conferência rápida de imagens no projeto.

Live Server

https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer

A extensão Live Server do VS Code cria um servidor local que atualiza automaticamente a página no navegador sempre que você salva um arquivo, acelerando o desenvolvimento web com feedback instantâneo.

Prettier Code Formatter

https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode

A extensão Prettier Code Formatter do VS Code formata automaticamente seu código conforme padrões definidos, garantindo estilo consistente e legível em várias linguagens, sem você precisar se preocupar com espaçamentos, quebras ou indentação manual.

Portuguese Brazilian Language Pack for VS Code

https://marketplace.visualstudio.com/items?itemName=MS-CEINTL.vscode-language-pack-pt-BR

Essa extensão traduz a interface do Visual Studio Code para português brasileiro, facilitando o uso do editor para quem prefere trabalhar em português.
